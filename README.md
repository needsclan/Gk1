Kør npm install
npm install expo-location
kør npx expo start --tunnel

Vi gemmer chat cv osv på realtimefirebase, og billederne bliver gemt på en rigtig database på firebase
Der er både kamera og søgefilter funktion med location til, appen er lavet baseret på undersøgelse af hvad folk finder vigtigt samt analyse af konkurrenter. De ville ha en hurtig måde at komme i kontakt uden at tage stilling til alle, derfor er swipe ned det samme som at add en på chat.
